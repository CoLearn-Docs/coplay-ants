# Raspberry Pi Zero 2 W Project

## Overview

This repository contains the code and supporting files for a project using the Raspberry Pi Zero 2 W.

## Hardware requirements
- Raspberry Pi Zero 2 W
- Micro SD Card (8GB or more) for uploading Raspbian image

## Software requirements
- Raspberry Pi OS (Raspbian)
- Python3 (Installed by default on Raspberry Pi OS)

## Setup
1. Flash Raspberry Pi OS onto the micro SD card using a tool such as the [Raspberry Pi Imager](https://www.raspberrypi.com/software/).
2. Insert the micro SD card into your Raspberry Pi Zero 2 W.
3. Connect your Raspberry Pi to a power source and boot it up.
4. After booting up, connect to the internet. You can either use a wired connection with an ethernet adapter or set up WiFi by following the instructions from [here.](https://www.raspberrypi.com/documentation/computers/configuration.html)
